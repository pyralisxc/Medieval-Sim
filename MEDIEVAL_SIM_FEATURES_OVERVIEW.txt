========================================================================
                     MEDIEVAL SIM MOD - FEATURE OVERVIEW
========================================================================

WHAT IS MEDIEVAL SIM?
---------------------
Medieval Sim is a comprehensive admin toolkit mod for Necesse that enhances 
server management and provides powerful tools for administrators and server 
owners. The mod focuses on creating a user-friendly experience for complex 
server administration tasks.

MOD PURPOSE: Transform complex console commands and admin tasks into intuitive 
visual interfaces that anyone can use without technical knowledge.

========================================================================
                           MAIN FEATURE AREAS
========================================================================

1. ADMIN TOOLS SYSTEM
======================

WHAT IT DOES:
- Provides a central access point for all administrative functions
- Adds an "Admin Tools" button to the game interface (only visible to admins)
- Creates a permission-based system that hides admin features from regular players
- Serves as the main hub that connects all other mod features

WHO CAN USE IT:
- Server administrators with ADMIN permission level
- Server owners

USER EXPERIENCE:
- Click the Admin Tools button in the inventory screen
- Opens a menu showing all available admin functions
- Clean, organized interface that groups related tools together
- No need to remember complex console commands

------------------------------------------------------------------------

2. BUILD MODE SYSTEM (Advanced Construction Tools)
==================================================

WHAT IT DOES:
- Provides enhanced building tools for administrators to quickly construct large structures
- Offers multiple placement patterns and shapes for efficient building
- Includes preview functionality so you can see what will be built before committing

CORE FEATURES:

Shape-Based Placement:
- Normal (single tile placement)
- Line (straight lines of any length)
- Cross (+ shaped patterns) 
- L-shaped patterns
- T-shaped patterns
- Square/Rectangle (filled or hollow)
- Circle (filled or hollow)
- Diamond patterns
- Half-circle patterns
- Triangle patterns

Configuration Options:
- Adjustable sizes (line length, square size, circle radius)
- Spacing between placed items
- Direction control (up, down, left, right)
- Hollow vs. filled shapes

PRACTICAL USES:
- Rapidly build walls, floors, and foundations
- Create geometric patterns for decorative purposes
- Build roads, paths, and infrastructure quickly
- Construct large buildings without tedious tile-by-tile placement

USER EXPERIENCE:
- Select your building tool and material
- Choose your desired shape and size from a visual menu
- See a preview of what will be placed
- Click to place the entire pattern at once

------------------------------------------------------------------------

3. ZONE MANAGEMENT SYSTEM
=========================

WHAT IT DOES:
- Creates designated areas with special rules and protections
- Allows administrators to define and manage different types of zones across the world
- Provides visual boundaries so players can see zone areas while in zones
- Enables fine-grained control over player interactions in different areas

ZONE TYPES:

Protected Zones:
- Areas where tiles and objects cannot be destroyed
- Prevents griefing in important areas like spawn points or community buildings
- Can be assigned to specific teams or players
- Perfect for protecting public infrastructure

PvP Zones:
- Combat Zones: Areas where PvP is specifically enabled
- Allows server owners to create mixed PvP/PvE experiences
- Allows admins to fine tune pvp damage multipliers, combat lock timers, and DoT multipliers
- Players get notifications when entering/leaving PvP zones

ZONE MANAGEMENT FEATURES:
- Create zones by selecting an area on the map
- Name zones for easy identification
- Edit zone boundaries after creation
- Delete zones when no longer needed
- View a list of all zones on the server
- Assign zones to specific teams or players

VISUAL FEATURES:
- Zone boundaries are visible to admins in zone management UI
- Different colors for different zone types
- Zone names displayed when near boundaries
- Particle effects to mark zone edges

PRACTICAL USES:
- Protect spawn areas from griefing
- Create safe trading posts
- Designate PvP arenas for organized combat
- Protect community projects and important structures
- Create roleplay areas with specific rules

------------------------------------------------------------------------

4. COMMAND CENTER SYSTEM
=========================

WHAT IT DOES:
- Takes Necesse's complex console commands and wraps them in user-friendly interfaces
- Eliminates the need to memorize command syntax, parameters, and options
- Provides visual forms with dropdowns, checkboxes, and input fields instead of text commands
- Makes advanced server management accessible to non-technical administrators

THE PROBLEM IT SOLVES:
- Necesse has many powerful console commands, but they're hard to use
- Commands require exact syntax and parameter knowledge
- Easy to make mistakes that could affect the server
- New administrators struggle to learn all the available commands

THE SOLUTION:
- Scans all available Necesse commands automatically
- Creates visual interfaces for each command
- Shows all available options in dropdowns and menus
- Validates input before sending commands
- Provides helpful descriptions for each option

COMMAND CATEGORIES:

World Management:
- Set time of day
- Change weather
- Teleport players
- Manage world settings

Player Management:
- Give items to players
- Set player permissions
- Manage player inventories
- Handle player statistics

Server Control:
- Server restart commands
- Configuration changes
- Performance monitoring
- Backup management

Item and Object Commands:
- Spawn items and objects
- Modify item properties
- Bulk item operations
- Inventory management

USER EXPERIENCE:
- Browse commands by category instead of searching through documentation
- Fill out forms instead of typing complex command syntax
- See all available options for each parameter
- Preview commands before executing them
- Get immediate feedback on results

TECHNICAL BENEFITS:
- Uses reflection to automatically discover new commands as Necesse updates
- Adapts to new command parameters without mod updates
- Reduces human error in command execution
- Provides consistent interface across all commands

------------------------------------------------------------------------

5. SETTLEMENT SPACING SYSTEM
============================

WHAT IT DOES:
- Increases the minimum distance required between player settlements
- Prevents settlements from being built too close together
- Helps reduce conflicts over territory and resources
- Encourages proper town planning and expansion

CURRENT BEHAVIOR:
- Adds 50 additional tiles of required spacing between settlements
- Prevents both new settlement creation and expansion if too close to existing settlements
- Maintains existing settlements (doesn't affect already-built settlements)
- Provides clear error messages when placement fails

BENEFITS:
- Reduces territorial disputes between players
- Encourages players to plan settlements more carefully
- Creates natural buffer zones between communities
- Prevents accidental blocking of settlement expansion
- Makes the world feel less crowded and more realistic

USER EXPERIENCE:
- When trying to place a settlement flag, get clear feedback if location is too close
- See helpful error messages explaining why placement failed
- Existing settlements continue to function normally
- No impact on gameplay for settlements that were already established

------------------------------------------------------------------------

6. PLOT FLAG SYSTEM
===================

WHAT IT DOES:
- Allows administrators to create purchasable settlement plots
- Enables controlled settlement expansion through an economy system
- Provides customizable pricing for different plot locations

HOW IT WORKS:

Plot Creation:
- Administrators purchase special "Plot Flag" items from the shop
- Plot flags cost configurable amount of coins (default: 1000 coins)
- Place plot flags in desired locations around the world
- Plot flags respect settlement spacing rules

Player Experience:
- Players find plot flags placed by administrators
- Right-click plot flag to see purchase price
- Spend in-game currency to buy the plot
- Plot flag automatically converts to settlement flag upon purchase
- Purchaser becomes the settlement owner

Administrative Control:
- Set coin cost via mod settings (ModConfig.PlotFlags.coinCost)
- Enable/disable plot flag system globally (ModConfig.PlotFlags.enabled)
- When enabled, normal settlement flags cannot be purchased/placed
- Plot flags respect settlement spacing configuration
- Remove or replace plots as needed

BENEFITS:
- Creates an economic aspect to settlement creation
- Allows administrators to control where settlements can be built
- Generates currency sinks for the server economy
- Enables strategic placement of settlements in desirable locations
- Prevents random settlement sprawl

------------------------------------------------------------------------

7. SETTLEMENT PROTECTION SYSTEM
================================

WHAT IT DOES:
- Allows settlement owners to protect their entire settlement from griefing
- Provides granular permission controls similar to Protected Zones
- Automatically applies protection to settlement boundaries
- Shows visual buff indicator when inside protected settlements

WHO CAN USE IT:
- Settlement owners (only the owner can configure protection)
- Settlement members (can be granted permissions by owner)
- Owner's team (can be granted full access)

HOW IT WORKS:

Enabling Protection:
- Settlement owner opens settlement settings (click settlement flag â†’ "Settlement" button)
- Click "Protection" button at top of settlement settings
- Toggle "Enable Protection" checkbox
- Configure permissions and team access
- Click "Save" to apply settings

Permission Controls:
- Team Permissions: Allow owner's team to build and interact
- Break Blocks: Allow breaking tiles
- Place Blocks: Allow placing tiles
- Doors: Allow opening/closing doors
- Containers: Allow accessing chests and containers
- Crafting Stations: Allow using crafting stations
- Signs: Allow editing signs
- Switches: Allow using switches
- Furniture: Allow interacting with furniture

Visual Feedback:
- Settlement Protection buff appears when entering protected settlement
- Buff shows settlement name, owner, and permission status
- Buff disappears when leaving settlement
- Clear indication of protection status

Integration:
- Works alongside Protected Zones (both systems can coexist)
- Respects settlement boundaries automatically
- No manual zone creation required
- Protection follows settlement as it expands

BENEFITS:
- Protects settlements without requiring manual zone creation
- Gives settlement owners control over their own protection
- Simpler than creating Protected Zones for every settlement
- Automatically adapts to settlement size changes
- Clear visual feedback for players

USER EXPERIENCE:
- Settlement owner clicks settlement flag
- Opens settlement settings
- Clicks "Protection" button
- Configures permissions in dialog
- Saves settings
- Protection immediately applies to entire settlement
- All players see buff when entering protected settlement

8. GUILD SYSTEM (In Development)
================================

WHAT IT DOES:
- Provides a complete guild/clan system for player communities
- Allows players to form groups with shared resources and permissions
- Offers hierarchical ranks with configurable permission levels
- Includes shared storage, research system, and teleportation network

GUILD STRUCTURE:

Rank Hierarchy (lowest to highest):
- Recruit: New members with limited permissions
- Member: Standard members with basic access
- Officer: Management role with elevated permissions
- Leader: Full control over guild settings

Permission Categories:
- Building & Breaking: Control who can modify terrain
- Storage Access: Guild bank and settlement storage permissions
- Territory: Claim and manage guild territory
- Treasury: Deposit/withdraw gold management
- Research: View, donate, and manage research priorities
- Membership: Invite, kick, and rank management
- Settings: Guild name, crest, tax rate, and permissions

GUILD FEATURES:

Guild Artisan NPC:
- Central hub for all guild interactions
- Create new guilds or join existing ones
- View guild information and member lists
- Edit guild crest with visual designer
- Access guild bank
- Manage research projects

Guild Bank:
- Shared inventory storage for guild members
- Multiple tabs for organization
- Deposit/withdraw permissions per rank
- Tracks contributions for fair distribution

Guild Research:
- Research projects that unlock guild benefits
- Resource contribution from members
- Progress tracking and completion rewards
- Research points system

Teleport Network:
- Guild Teleport Banners for fast travel
- Waving banner animation (visual appeal)
- Teleport between any guild banners
- Auto-claims to first guild member's guild

Visual Identity:
- Custom guild crests with:
  - Background shapes
  - Primary and secondary colors
  - Emblems and symbols
  - Border styles
- Guild Banners (placeable/wearable)
- Guild Crests (wearable trinket)

Guild Objects:
- Guild Flag: Decorative flag showing guild colors
- Guild Chest: Shared storage container
- Guild Teleport Banner: Fast travel network
- Guild Cauldron: Guild-wide potion brewing (planned)

GUILD MANAGEMENT:

Creating a Guild:
1. Find a Guild Artisan NPC
2. Click "Create Guild"
3. Enter guild name
4. Choose public or private visibility
5. Guild is created with you as Leader

Joining a Guild:
- Public guilds: Browse and join directly
- Private guilds: Receive and accept invitations
- Only one guild membership at a time

Member Management:
- Leaders can promote/demote members
- Officers can invite new members
- Officers can kick lower-ranked members
- Audit log tracks all membership changes

INTEGRATION WITH OTHER SYSTEMS:
- Zone Management: Guilds can own protected zones
- Settlement Protection: Guild territories can be protected
- Grand Exchange: Guild treasury can fund purchases
- Build Mode: Officers can use build tools in guild areas

HOW EVERYTHING WORKS TOGETHER:
------------------------------

1. Administrator logs in and sees the Admin Tools button
2. Clicks Admin Tools to open the main menu
3. Chooses from available options:
   - Build Mode: For construction projects
   - Zone Management: To protect areas or set up PvP zones
   - Command Center: To use advanced Necesse commands easily
   - Settlement Tools: To manage settlement spacing and plots

4. Each tool opens its own specialized interface
5. All tools work together - for example:
   - Use Build Mode to construct a building
   - Use Zone Management to protect it from griefing (or use Settlement Protection for player settlements)
   - Use Command Center to add special items or effects
   - Use Settlement tools to control nearby settlement creation
   - Settlement owners can protect their own settlements without admin intervention

PERMISSION SYSTEM:
- All features respect Necesse's permission levels
- Only administrators and server owners can access admin tools
- Regular players see normal game interface
- Different permission levels can be set for different features

VISUAL CONSISTENCY:
- All interfaces follow the same design language
- Consistent button layouts and menu structures  
- Clear visual feedback for all actions
- Helpful tooltips and descriptions throughout
